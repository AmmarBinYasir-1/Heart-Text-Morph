# 3D Particle Text Morph

Interactive 3D particle animation built with **Three.js**.  
Particles morph smoothly between a **heart shape** and **user typed text** in real time.

## ✨ Features

- 40,000 animated particles
- Morphing between **heart shape** and **custom text**
- Mouse interaction with particle repulsion
- Smooth physics-based animation
- WebGL rendering using **Three.js**
- Clean minimal UI
- Translation disabled (English only)
